# Курсова робота з дисципліни **«Штучні нейронні мережі»**.

**НТУ "Харківський політехнічний інститут"**

**Виконав: Касвінов Віктор, КН-1224с**

**Розпізнавання рукописних цифр (MNIST)**

## Опис проєкту
Цей проєкт присвячений створенню згорткової нейронної мережі (CNN) для класифікації рукописних цифр на основі популярного набору даних **MNIST**. Проєкт включає етап навчання моделі «з нуля» та розробку інтерактивного веб-застосунку для перевірки результатів у реальному часі.

## Функціонал
- **Навчання CNN:** Модель з використанням шарів BatchNormalization та Data Augmentation.
- **Веб-інтерфейс:** Застосунок на Streamlit, що підтримує:
    - Завантаження власних зображень цифр.
    - Малювання цифри курсором на інтерактивному полотні.
    - Автоматичне центрування та обробку вводу за допомогою OpenCV.
    - Візуалізацію розподілу ймовірностей прогнозів.

## Технологічний стек
- **Мова:** Python 3.10+
- **Deep Learning:** TensorFlow, Keras
- **Обробка зображень:** OpenCV, Pillow
- **Web UI:** Streamlit
- **Аналіз даних:** NumPy, Scikit-learn, Matplotlib, Seaborn

## Встановлення та запуск

1. **Клонуйте репозиторій:**
   ```bash
   git clone https://github.com/REQUIEM-0G/CourseProject.git
   cd CourseProject
   ```

2. **Встановіть залежності:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустіть веб-застосунок:**
   ```bash
   streamlit run src/app.py
   ```
